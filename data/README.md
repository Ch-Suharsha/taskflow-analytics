# üìÅ Data Directory ‚Äî Schema Documentation

All CSV files are generated by `python/data_generation.py`. Below is the schema for each table.

## `users.csv` (10,000 rows)
| Column | Type | Description |
|--------|------|-------------|
| user_id | string | Unique user identifier (U000001) |
| workspace_id | string | Workspace the user belongs to |
| email | string | User email |
| signup_date | datetime | Account creation date |
| account_tier | string | free / starter / professional / enterprise |
| user_role | string | owner / admin / member / guest |
| industry | string | Company industry |
| team_size | int | Self-reported team size |
| signup_source | string | organic / paid_search / referral / sales |
| country | string | User country |
| is_active | bool | Whether user is currently active |
| last_login_date | datetime | Most recent login |

## `user_activity_summary.csv` (10,000 rows)
| Column | Type | Description |
|--------|------|-------------|
| user_id | string | FK to users |
| workspace_id | string | FK to workspaces |
| signup_date | date | Account creation date |
| days_since_signup | int | Days between signup and analysis date |
| total_sessions | int | Total login sessions |
| total_events | int | Total behavioral events |
| avg_session_duration_min | float | Average session length |
| tasks_created | int | Tasks created by user |
| tasks_completed | int | Tasks completed by user |
| boards_created | int | Boards created |
| premium_features_used | int | Count of premium features used |
| last_active_date | date | Last activity date |
| is_power_user | bool | Top 20% by activity |
| is_at_risk_churn | bool | No login in 14+ days |
| cohort_month | string | Signup month (YYYY-MM) |

## `onboarding_funnel.csv` (10,000 rows)
| Column | Type | Description |
|--------|------|-------------|
| user_id | string | FK to users |
| step_1_completed | bool | Completed "Team Setup" |
| step_2_completed | bool | Completed "Invite Members" |
| step_3_completed | bool | Completed "Create Board" (**36% drop-off**) |
| step_4_completed | bool | Completed "Create Task" |
| onboarding_completed | bool | Completed all 4 steps |
| ab_test_variant | string | control / variant_a / null |

## `feature_usage.csv` (~18K rows)
| Column | Type | Description |
|--------|------|-------------|
| usage_id | string | Unique record ID |
| user_id | string | FK to users |
| feature_name | string | Feature identifier |
| first_used_date | datetime | First usage timestamp |
| total_usage_count | int | Total times used |
| last_used_date | datetime | Most recent usage |
| days_since_signup_at_first_use | int | Discovery delay |

## `subscriptions.csv` (~2,500 rows)
| Column | Type | Description |
|--------|------|-------------|
| subscription_id | string | Unique sub ID |
| workspace_id | string | FK to workspaces |
| plan_type | string | free / starter / professional / enterprise |
| mrr | float | Monthly recurring revenue |
| is_active | bool | Currently paying |
| churn_reason | string | price / feature_gap / competitor / other |

## `ab_test_assignments.csv` (~1,200 rows)
| Column | Type | Description |
|--------|------|-------------|
| user_id | string | FK to users |
| test_name | string | Experiment identifier |
| variant | string | control / variant_a |
| converted | bool | Completed onboarding |

## `events.csv` (~500K rows, sampled)
| Column | Type | Description |
|--------|------|-------------|
| event_id | string | Unique event ID |
| user_id | string | FK to users |
| event_name | string | Action type (task_created, etc.) |
| event_timestamp | datetime | When event occurred |
| session_id | string | Session grouping |
